# Contributing

Thanks for your interest in improving Hikari-Tweaks.

## Setup

1. Install Java 17.
2. Clone this repository.
3. Run `./gradlew build`.

## Pull Requests

1. Create a branch from `main`.
2. Keep changes focused and small when possible.
3. Include a short summary of what changed and why.
4. Ensure the project builds before opening a PR.

## Bug Reports

Please use the GitHub issue template and include:

- Mod version
- Minecraft / Fabric Loader version
- Steps to reproduce
- Logs or crash reports
